# 깃헙 레포지토리 조회 앱

### 사용 기술
- [Retrofit](https://square.github.io/retrofit/)
    - OKHttp 를 네트워크 레이어로 활용하는 상위호환 라이브러리

- [Github Open API](https://docs.github.com/ko/rest)
- [RecyclerView](https://developer.android.com/guide/topics/ui/layout/recyclerview)
- ListAdapter
- Handler
